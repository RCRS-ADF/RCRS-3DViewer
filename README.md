# RCRS-3DViewer
## How to launch
Please initialize config using the script when run first time.
```Shell
./setup_config.sh
```

Launch viewer
```Shell
java -jar rcrsviewer3d.jar
```

## key bind
- Zoom out
  - Page Up
- Zoom in
  - Page Down
- Camera angle
  - [arrow keys]
- Camera location
  - W
  - A
  - S
  - D
